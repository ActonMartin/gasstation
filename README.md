## 项目介绍
本项目是一个用于获取指定省份城市加油站信息的工具，通过调用腾讯位置服务API，将加油站信息整理并保存到Excel文件中。支持多城市查询，自动去重，且能自动调整Excel列宽。

## 配置步骤
1. **注册腾讯位置服务账号**：访问 [腾讯位置服务官网](https://lbs.qq.com/) 注册并登录账号。
2. **获取API密钥**：在控制台创建应用，获取 `KEY`。
3. **配置项目**：
   - 复制 `config_template.py` 为 `config.py`。
   - 在 `config.py` 中设置 `KEY` 为你在腾讯位置服务获取的密钥。

## 环境准备
本项目使用Python 3.x，所需依赖可通过以下命令安装：
```bash
pip install -r requirements.txt
```

## 使用方法
1. **修改城市列表**：在 `main.py` 中修改 `locations` 列表，添加需要查询的省份和城市信息。示例：
```python
locations = [
    {'province': '浙江省', 'city': '丽水市'},
    {'province': '广东省', 'city': '深圳市'}
]
```
2. **运行程序**：在项目根目录下运行以下命令：
```bash
python main.py
```
3. **查看结果**：程序运行完成后，加油站信息将保存到以省份命名的文件夹下的Excel文件中。

## 注意事项
- 本项目使用频率限制器，默认每秒最多5次请求，避免触发API频率限制。
- 若请求失败，程序会输出错误信息并跳过当前页面。
- Excel文件会自动去重，同一加油站ID只记录一次。
